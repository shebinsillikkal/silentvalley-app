# Silent Valley — Nature & Trekking App 🌿

> The companion Android app for **Silent Valley National Park**, Kerala — one of India's last remaining pristine rainforests.

## About
Silent Valley National Park in Palakkad, Kerala is one of the most biodiverse places on earth — untouched shola forests, endangered species, and some of the best trekking in South India. This app was built to help visitors explore it safely, responsibly, and with full information.

## Features
- 🗺️ **Offline Trail Maps** — downloadable maps that work without internet inside the forest
- 🦜 **Wildlife Guide** — 400+ species with photos, sounds & conservation status
- 📅 **Visit Booking** — book entry permits, guided tours & trekking slots
- 🚶 **Trek Tracker** — live GPS tracking with trail markers & waypoints
- 📷 **Sightings Log** — record wildlife sightings with location & photo
- 🌤️ **Weather & Alerts** — real-time weather + forest closure alerts
- ℹ️ **Visitor Info** — entry rules, timings, do's & don'ts, emergency contacts
- 🌱 **Conservation Feed** — latest news from the forest & wildlife updates

## Tech Stack
```
Flutter 3.x | Dart | Firebase | Google Maps | Offline Maps (mbtiles)
Firebase Auth | Firestore | FCM | Razorpay (booking payments)
```

## Screens
| Screen | Description |
|--------|-------------|
| Home | Weather, today's alerts, quick booking, recent sightings |
| Trail Map | Interactive offline map with all trails, zones & viewpoints |
| Wildlife | Species guide — mammals, birds, reptiles, plants |
| Book Visit | Select date, zone, number of visitors & pay online |
| My Trek | Live GPS tracking during trek, emergency SOS button |
| Sightings | Log wildlife sightings with photo, GPS & notes |
| Info | Rules, timings, contact numbers, forest history |

## Conservation Note
This app is designed to promote **responsible eco-tourism**. All sighting data collected (with user consent) is shared with the Forest Department for conservation research.
